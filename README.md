## 🇻🇳 Tìm hiểu cách đẩy dự án lên GitHub
Trong quá trình thực hiện dự án, tôi đã tìm hiểu và thực hành cách sử dụng Git để quản lý mã nguồn và đẩy (push) dự án lên GitHub. Việc này giúp lưu trữ mã nguồn trực tuyến, theo dõi lịch sử thay đổi và dễ dàng chia sẻ dự án với người khác.

## 🇺🇸 Learning how to push a project to GitHub
During the development of this project, I learned and practiced how to use Git to manage source code and push the project to GitHub. This helps store the source code online, track changes, and easily share the project with others.
